BUILT BEYOND THE BASICS

작은 차이가
긴 대화를 편하게.

보이는 통화 품질

RTT, 지터, 손실, 코덱을 확인하고 끊어진 오디오를 자동으로 복구합니다.

필요한 소리만 공유

Windows에서는 전체 컴퓨터뿐 아니라 선택한 프로그램의 소리만 골라 공유할 수 있습니다.

팀에 맞는 권한

역할과 사용자별로 접근, 작성, 음성, 사운드, 화면 공유 권한을 나눕니다.

내가 소유하는 서버

Node.js 18+만으로 서버를 열고, 필요할 때 TURN을 더해 외부 연결을 안정화합니다.

채널 밖에서도 DM

고유 사용자 코드로 1:1 대화를 시작하고 중요한 맥락을 이어갑니다.

흐름을 지키는 집중 모드

사이드바를 접고 지금 쓰는 채팅, 메모, 그림판만 화면 가득 펼칩니다.
