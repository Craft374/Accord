# Accord 패치노트

## 2.0.1 (2026-07-14)

### 버그 수정
- **다이렉트 메시지 `+` 버튼 모양 교정** — 전역 버튼 스타일(`min-height:36px`, 좌우 여백)이 적용돼 26×26 정사각형이어야 할 버튼이 세로로 늘어난 직사각형(28×36)으로 찌그러져 `+` 아이콘이 삐딱하게 보이던 문제 수정. `padding`·`min-height`를 초기화해 정사각형 + 아이콘 정중앙으로 복구.
- **파일 선택 버튼 상하 간격 불균형 수정** — 파일 입력(채널 아이콘·프로필 이미지 등)의 "파일 선택" 버튼이 위아래 여백이 달라 위쪽으로 쏠려 보이던 문제 수정. 입력 요소의 잔여 테두리·여백을 제거하고 버튼 높이를 고정해 상하 4px로 균등하게 중앙 정렬.

## 2.0.0 (2026-07-13)

### 기능 추가
- **권한 시스템 개편**
  - 특정 유저 권한 칸을 `허용 ↔ 거부` 2단계 토글로 단순화(상속 단계 제거)하고, 행 오른쪽에 `×` 삭제 버튼 추가로 유저 권한을 한 번에 제거 가능.
  - 통화방에 세부 권한 신설 — `마이크·스피커(듣기)`, `소리 공유`, `화면 공유`를 역할/유저별로 각각 허용·거부 가능. 권한이 없으면 해당 버튼이 숨겨지고, 마이크·스피커 권한이 없으면 자동 음소거 및 상대 소리 차단.
- **어코드 로고 적용** — 좌측 상단과 로그인 화면의 `A` 표시를 실제 Accord 아이콘으로 교체.
- **DM 아이콘 개선** — 채널 목록의 다이렉트 메시지(편지봉투) 아이콘을 더 크고 선명하게 교체.
- **서버 접속 화면 문구 정리** — 서버 주소 입력 화면의 `Accord v2` → `Accord`.

### 버그 수정
- **유휴 상태 연결 끊김 수정** — 아무 조작 없이 가만히 두면 잠시 뒤 서버 연결이 끊기던 문제 수정. 서버가 주기적으로 연결 유지 신호를 보내고, 끊겨도 클라이언트가 자동으로 다시 연결.
- **채널 아이콘 마우스 오버 깨짐 수정** — 아이콘이 있는 채널에 마우스를 올리면 이미지가 좌측 상단으로 튀며 잘려 보이던 문제 수정.
- **버튼/파일 선택 UI 정렬 개선** — 삐딱하던 아이콘 버튼과 위로 쏠리던 파일 선택 영역 정렬 개선.
